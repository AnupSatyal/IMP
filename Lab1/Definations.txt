HTML:
HTML stands for "Hypertext Markup Language". HTML is used to create websites/webpages. Tags are used to define the layout of the page. It is the basic structure of the page, the basic front-end of the website, the button tase, input forms are saved in HTML element.

CSS:
It stands for "Cascading Style Sheet", mostly they are used to style HTML element. CSS can be directly written into HTML pages or creating separate .css file. They are used to define text styles, tables, font size ,colour etc. of the page.

JavaScript:
JavaScript is one of the most and commonly used programming language, mainly used in web development. It is used in both client-side and sever-side. JavaScript functions are tagged in script, they are responsible for functionality of the webpage.

http:
It stands for "Hypertext Transfer Protocol." It is a protocol used to transfer data over the web. When client request or access the webpage, the browser sends the request to servers and responds if the URL is valid or not, and redirect files if it is valid.

html5:
It is latest and updated version of HTML. With the the latest version many new and different tags are introduced.

DHTML:
It's a Dynamic HTML, used when the collaborative programming language are used to make one animated webpage, the programming language can be HTML, CSS, JavaScript and DOM.

Coding Conventions:
It is the set of rules to write the codes in programming. The rules refers to the style, practice, methods etc. for each aspect of program.
It makes the code easier to understand and it is very important to maintain the code and applicaction standard.
